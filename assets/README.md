# Extension artwork

`extension-icon.svg` is a 600 × 300, 2:1 gallery image for the Asset Manager extension.

The illustration uses an original dog silhouette with a floppy ear, extended muzzle, and raised tail. It intentionally avoids the Scratch Cat silhouette while preserving the idea of applying cached assets to a sprite-like character and the stage.

The artwork is composed entirely of editable SVG geometry and contains no text.
